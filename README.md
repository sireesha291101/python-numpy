NumPy Practice Programs

This repository contains NumPy programs created for learning and practicing array operations in Python.

Topics Covered

- Creating NumPy arrays
- Array shape and data types
- Array indexing
- Array slicing
- Using "arange()"
- Using "eye()"
- Reshaping arrays
- Flattening arrays
- 2D and 3D arrays
- Element-wise multiplication
- List comprehensions
- Array traversal and manipulation
- view() and copy()
- hstack and vstack
- adding, inserting, deleting the elements

Sample Programs

- Create 1D, 2D, and 3D arrays
- Find array shape and data type
- Generate arrays using "arange()"
- Create identity matrices using "eye()"
- Reshape arrays into different dimensions
- Flatten multi-dimensional arrays
- Perform array slicing and indexing
- Multiply corresponding elements of arrays

Learning Objectives

- Understand NumPy fundamentals
- Work with multi-dimensional arrays
- Perform efficient array operations
- Build a foundation for Data Science and Machine Learning

Language: python-numpy
